# CV en Ligne - Nicole Benty Gomis

## Description

Ce projet est un CV en ligne développé en HTML, CSS et JavaScript dans le cadre d'un devoir académique. Il présente mon parcours académique, mes compétences, mes expériences professionnelles, mes projets réalisés ainsi que mes informations de contact.

## Fonctionnalités

* Présentation personnelle
* Informations personnelles
* Parcours académique
* Compétences techniques avec barres de progression
* Compétences personnelles
* Expériences professionnelles
* Projets réalisés
* Langues parlées
* Centres d'intérêt
* Formulaire de contact
* Mode sombre
* Bouton retour en haut de page
* Menu responsive pour les petits écrans
* Mise à jour automatique de l'année dans le pied de page
* Validation du formulaire de contact
* Animations interactives en JavaScript

## Technologies utilisées

* HTML5
* CSS3
* JavaScript
